# 🧩 Laravel Task Manager REST API

A simple RESTful Task Management API built using Laravel.  
This project demonstrates CRUD operations, validation, and MVC architecture.

---

## 🚀 Tech Stack

- PHP 8+
- Laravel Framework
- MySQL / PostgreSQL
- REST API (JSON)
- Postman (for testing)

---

## 📌 Features

- Create task (POST)
- View all tasks (GET)
- Update task (PATCH)
- Delete task (DELETE)
- Input validation
- RESTful API structure
- MVC architecture (Model - View - Controller)

---

## 📁 Project Structure

- `routes/api.php` → API routes
- `app/Http/Controllers` → Business logic
- `app/Models` → Database models
- `database/migrations` → Database schema

---

## 📡 API Endpoints

### Get all tasks

GET /api/tasks

### Create task

POST /api/tasks

### Update task

PATCH /api/tasks/{id}

### Delete task

DELETE /api/tasks/{id}
